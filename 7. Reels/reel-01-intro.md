# Reel #1 — Intro / Origin Story

## Script

I challenged myself to build 100 free solutions for 100 different people.

Project number one? A project for myself.

I wanted to start building something, so I started learning to code and build things with AI. And then I thought — if I'm going to learn this anyway, I might as well do it for real people with real problems.

So I built this, my own page in case anyone would be interested in joining the challenge. Now anyone can come and tell me their problem. If I think I can actually help, I build the solution, with my own resources and time.

Got a problem worth solving? Link's in the bio.
